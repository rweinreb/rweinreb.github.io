Abstract

This is a document for a course project in COMP 305 Software Engineering I, Fall 2019. This software will help HBF International with their fruit intake, fruit processing, and payments to growers. This will be a technology platform that will allow any user to download the application and run it on their device. Each user will have their own login and will have access to various parts of the software. Users will be able to create new purchase orders, add pallet, remove pallet, create new jobs, add package type, add grower charge, add price to grower, and print settlement.
