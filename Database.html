Database Specifications

Employees

This is a file of all the employees working. This will have every employee number. It wall track Purchase Orders entered, Packing Orders made, Shipment Orders, and Checks. This will allow the company to track each employees movements.

Purchase Orders

This is a file in the system that has all of the grower information, and all of the fruit they have delivered. Employees will use this file to track fruit, make packing orders, and pay growers. Each order will have a unique number that employees will use as a reference.

Packing Order

This file will have every packing order made. The Purchase Order will be included as well as the Sales Order. Employees can look back on these files to ensure that all product is being used efficiently.

Sales Order

This file will have all sales orders made. The customer and quantity will be listed to easily track different orders. Employees will use this file to ensure each order is completed on the date they need.

Shipment Order

This file has all Shipment Orders sent out. It will show the Sales Order Number and Purchase Order number. This will allow the employees to track fruit back to the growers. This will help A/P when they pay the growers.

Checks to Grower

This file keeps a record of all checks sent to growers. It will have each Purchase Order listed as well as each employee. At the end of the season employees will use this to make a budget for the next year. Employees can also periodically check this to ensure payments are being made.
